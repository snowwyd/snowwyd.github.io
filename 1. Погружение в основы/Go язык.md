*Цель: поставить себе Go, написать в нем простейшие программы с использованием основного синтаксиса*

Твоя задача посмотреть вот это видео про основы Go:
https://youtu.be/d91sOEh1vfc
P.S. текстовую версию курса из его тгк тоже советую бегло просмотреть

Прогресс:
- [ ] Поставить Golang
- [ ] Поставить IDE
- [ ] Посмотреть видео и написать код из него самому
- [ ] Написать несколько простых программ на основе тех тем, которые были пройдены
- [ ] Написать unit-тесты, проверяющие то, как ведут себя написанные ранее функции
- [ ] Попробовать написать программу, которая запустит функцию в двух горутинах
- [ ] Написать программу с одним каналом

*Результат: теперь ты уже можешь написать какую-то простую логику на Go, а это значит, что можно пробовать что-то уже реально интересное и приближенной к реальной разработке*
